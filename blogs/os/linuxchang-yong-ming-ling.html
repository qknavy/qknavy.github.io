Linux常用命令

Navy 2020-06-17 linux

都是基本用法，孰能生巧，加油~

# 1、查看端口对应的进程号

netstat -nlp | grep :8081| awk '{print $7}' | awk -F"/" '{ print $1 }'

查看某个端口对应的进程号，并杀死进程：

kill netstat -nlp | grep :8081| awk '{print $7}' | awk -F"/" '{ print $1 }'

# 2、查看进程名称对应的进程id

pgrep -f [进程名称]
1

eg：杀死所有phantomjs进程

kill $(pgrep -f phantomjs)
1

# 替换文件内容

sed -i 's/被替换内容/用来替换的内容/g' 文件名

# mac下
sed -i "" "s/被替换内容/用来替换的内容/g" 文件名

# g-表示全局替换
